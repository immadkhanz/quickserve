# QuickServe 🍔

Welcome to the **QuickServe** repository! This is a comprehensive, full-stack restaurant management system built as part of my Software Engineering Internship (Intern ID: ZYNVEX-CERT-0613). 

This project is designed to modernize restaurant operations with a focus on real-time data, premium user experience, and robust role-based security.

## 📅 Internship Project Timeline & Modules

To ensure a structured and professional development cycle, I have divided this project into four weekly modules. This repository will be updated sequentially as each module is completed.

### Week 1: Foundation & Architecture (Module 1)
**Focus:** Project initialization and core design system.
*   Setup Next.js App Router and environment configurations.
*   Integrate Supabase Server-Side Rendering (SSR) utilities.
*   Design and implement a premium dark-mode, glassmorphic global CSS system.
*   Build the animated Splash Landing Page.
*   Implement secure, role-based login routing logic.

### Week 2: Advanced Customer Menu (Module 2)
**Focus:** Customer-facing interface and dynamic data rendering.
*   Redesign the Menu Card component with modern aesthetics.
*   Create an interactive category sidebar and live search functionality.
*   Integrate dynamic Supabase data fetching into the Menu Page.
*   Implement the Cart Drawer state and workflow.
*   Optimize UI for mobile responsiveness and cross-browser compatibility.

### Week 3: Real-Time Order Tracking (Module 3)
**Focus:** Real-time WebSockets and secure customer checkouts.
*   Design the Order Tracking page layout.
*   Integrate Supabase Realtime WebSockets for a live Estimated Time of Arrival (ETA) countdown.
*   Develop a 5-star digital receipt and customer complaint form.
*   Enforce strict edge middleware to protect admin/kitchen routes.
*   Finalize the end-to-end customer checkout flow.

### Week 4: Staff Control Centers (Module 4)
**Focus:** Admin dashboard and Kitchen Display System (KDS).
*   Redesign the Admin sidebar navigation and secure sign-out flow.
*   Implement a real-time admin revenue and analytics overview.
*   Build a full CRUD (Create, Read, Update, Delete) panel for menu management.
*   Create an admin inbox for reviewing customer feedback and flagging complaints.
*   Update the Kitchen KDS portal to calculate and broadcast live order ETAs.

---

## 🛠️ Tech Stack
*   **Frontend:** Next.js 15 (App Router), React 19, Tailwind CSS
*   **Backend / Database:** Supabase (PostgreSQL)
*   **Authentication:** Supabase Auth with SSR integration
*   **Real-time:** Supabase WebSockets (`pg_changes`)
